Exercício Fixação 01

Tags:

<html>:
- Tag responsável pela abertura de um documento HTML.
<head>:
- Tag responsável pela informação de configuração de uma página HTML.
<title>:
- Tag responsável pela informação do título de uma página HTML, fica contida dentro da tag <head>.
<body>:
- Tag responsável pelo conteúdo de uma página HTML.
<p>:
- Representa um parágrafo.
<h1>:
- Representa um título com maior grau de importância.
<h2 à h5>:
- Representam títulos com grau intermediário de importância.
<h6>:
- Representa um título com menor grau de importância.
<br>:
- Tag responsável pela adição de quebra de linha na página. Tag sem fechamento.
<img>:
- Tag responsável pela adição de imagem na página, sendo necessária a informação de localização do recurso por meio do atributo 'src'. Tag sem fechamento.
<input>:
- Tag responsável pela adição de elementos de preenchimento a partir da interação com o usuário. O tipo do elemento de preenchimento varia de acordo com o o atributo 'type'.
<a>:
- Tag responsável pela adição de âncoras na página HTML. As âncoras podem estar relacionadas a números de telefone, urls, emails, etc.
<ol>:
- Tag responsável pela adição de lista ordenada na página.
<ul>:
- Tag responsável pela adição de lista não ordenada na página.
<li>:
- Tag responsável pela adição de itens de lista na página, sendo parte integrante de uma lista ordenada ou não ordenada.
<code>:
- Tag que fará a representação visual de blocos de código em uma página HTML.
<dl>:
- Tag que representa na página HTML um par conceito-definição.
<dt>:
- Tag que representa na página HTML o conceito/titulo em uma <dl>.
<dd>:
- Tag que representa na página HTML a definição/descrição em uma <dl>.
<q>:
- Tag que representa uma citação curta na página HTML.
<blockquote>:
- Tag que representa uma citação na página HTML a qual, por meio do atributo 'cite', estará linkada à fonte da citação.
<abbr>:
- Tag que é responsável pela definição de uma sigla ou abreviação, a qual será exibida ao descansar o mouse sobre o termo. A definição será definida através do atributo 'title'.
<address>:
- Tag responsável pela formatação de um endereço.
<strong>:
- Tag que dá destaque a um termo ou texto de uma página HTML.
<b>:
- Tag que deixa em negrito um termo ou texto de uma página HTML.
<em>:
- Tag que dá ênfase a um termo ou texto de uma página HTML.
<i>:
- Tag que deixa em itálico um termo ou texto de uma página HTML.
<pre>:
- Tag responsável pela representação de um texto pré-formatado em uma página HTML.
<div>:
- Tag responsável pela representação de um container não semântico em bloco, em uma página HTML.
<span>:
- Tag responsável pela representação de um container não semântico em linha, em uma página HTML.
